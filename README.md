## Description
Created a command-line application that dymanically generates a professial README.md file from a user's input using the Inquirer package. The application is invoked using the command node index.js. Once started user is promoted to enter project title whichi will be title in the README. User is then promoted for a description, installation instructions, usage information, contribution guidelines, and test instuctions then this info is added to the sections of the README entitled Description, Installation, Usage, Contributing, and Tests. When user chooses a license then a badge is added near the top of the README and a notice is added to the section of the README License section with the link to the license information. When user enters github username its added to the Questions section with a link to their github profile. When user enters email address its added to Questions section with instructions on how to reach them with additional questions. Clicking each link in the table of contents will take you to the corresponding section of the README.
